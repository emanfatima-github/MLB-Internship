# Student Performance Dashboard

## Data Cleaning

During this project, I cleaned the dataset by checking for missing values and filling them where needed.
I removed duplicate records to avoid repeated data, renamed columns to make them easier to understand,
created a new "Average_Score" column by calculating the average marks of all subjects, and added
a "Performance" column to classify each student's performance as "Excellent", "Good", "Average", or "Needs Improvement".

## Data Visualizations

I created several visualizations to better understand the data:

* "Bar Chart" to compare the average score of each student.
* "Histogram" to see how the average scores are distributed.
* "Scatter Plot" to observe the relationship between Python and Machine Learning marks.
* "Pie Chart" to show the percentage of students in each performance category.
* "Box Plot" to visualize the spread of marks across all subjects and identify any unusual values.

## Key Insights

1. The top-performing students achieved consistently high average scores across all subjects.
2. Most students were in the "Good" or "Average" performance categories, while only a few needed improvement.
3. By comparing the average marks of all subjects, it was easy to identify which subject had the highest class
average and which subject may require more attention from students.
