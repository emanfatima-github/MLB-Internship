# Student Performance Analysis

## What I Learned about NumPy

* I learned that NumPy is a Python library used for numerical computing.
* I learned how to create 1D and 2D arrays.
* I practiced array indexing, slicing, reshaping, and performing mathematical operations like finding the sum, mean, maximum, and minimum values.

## What I Learned about Pandas

* I learned that Pandas is used for working with datasets.
* I learned how to read a CSV file using `read_csv()`.
* I learned how to filter data, calculate averages, and save the analyzed data as a new CSV file.

## Key Insights from the Dataset

* I calculated the average marks for each subject.
* I identified the top 5 performing students based on their total marks.
* I found the students who scored below the average total marks.
* I also found the total number of students in the dataset.
