# Computer Vision Image Processing Studio

## Project Overview

Computer Vision Image Processing Studio is a Gradio-based web application that performs different image processing operations using OpenCV. 
Users can upload an image, choose an operation from the dropdown menu, and instantly view the processed result through a simple and interactive interface.

The application is deployed on Hugging Face Spaces, making it accessible online without requiring users to install any software.

## Features

* Upload an image
* Display the original image
* Select an image processing operation
* Display the processed image
* Download the processed image
* Simple and user-friendly interface

## Image Processing Operations

* Grayscale
* Blur
* Edge Detection
* Image Rotation
* Image Enhancement
* Contour Detection
* Shape Detection

## Technologies Used

* Python
* OpenCV
* NumPy
* Pillow
* Gradio
* Hugging Face Spaces


## How to Use

1. Launch the application.
2. Upload an image.
3. Select an image processing operation from the dropdown menu.
4. View the processed image.
5. Download the processed image if required.

## Output

The application displays:

* Original Image
* Processed Image

Users can download the processed image directly from the interface.
